# Siphon Spec v0.1

Initial version with simple JSONPath extraction.

## Features
- Basic JSONPath syntax: `$.data.field`
- Request path configuration

## Example

```yaml
version: "0.1"

request:
  path: "/products"

extract:
  id: "$.data.id"
  name: "$.data.name"
  price: "$.data.pricing.amount"
```

## Output

```json
{
  "id": "prod_123",
  "name": "Widget",
  "price": 100
}
```
