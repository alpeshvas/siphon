# Siphon Spec v0.2

Added array iteration.

## Features
- Everything from v0.1
- Array iteration with `[*]` syntax

## Example

```yaml
version: "0.2"

request:
  path: "/products"

extract:
  id: "$.data.id"
  first_item_name: "$.data.items[*].name"
```

## Output

```json
{
  "id": "prod_123",
  "first_item_name": "Widget"
}
```

## Notes
- `[*]` returns the first item in the array
- Remaining path after `[*]` is applied to each item
